Example 1 - Controllers with views coming from templates

This is an instance of Example1Controller. Its parent class is also TemplateController so its content comes from a file that follows the convention of lowecase the name minus the 'Controller' suffix. In this case, the view of this controller will be the root element of the html you see in the example1.html file

You'll find that its content was loaded using requirejs using its text plugin
